0. 자사몰에서 사용자가 특별하고 고급스러운 경험을 할 수 있으면 좋겠다는 니즈

1. Three.js 사용
  ㄴ 기하와 벡터 공부

2. GLB 파일 필요 ( for 3D ) + 환경 hdri파일 [ https://polyhaven.com/ ]
   ㄴ 주변광 + 환경광의 역할을 하며 재질, 텍스처, 스킨 등을 실제 눈으로 보는 것과 유사하게 환경 세팅해줌


3. GLB 파일 생성을 위한 AI 활용 
   ㄴ Meshy AI 
       --> 설명적인 텍스트 프롬프트를 기반으로 텍스처(질감)을 생성하는 Text to Texture와 
            특정 이미지에서 질감을 가져오는 Image To Texture 기능을 제공하는 기능을 제공 함으로서
            이는 아티스트, 디자이너, 개발자의 워크플로를 크게 가속화할 수 있도록 설계된
            3D 생성 AI 도구 상자입니다.
    ㄴ  Blender 및 Unity용 API 서비스와 플러그인도 제공

    && AI를 활용하지 않고 GLB 파일 생성시 어떠한 프로세스를 거쳐야 하는지도 설명
